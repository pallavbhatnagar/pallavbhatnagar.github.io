UI Kit — made in Adobe XD

Social Network for travellers

A collaborative project with a friend — a set of ready-to-use components, layouts, and templates that product designers can directly use or even copy and paste into their user interfaces for quickly creating better products. It works like fast tools that can help you hasten the entire design workflow without drawing every UI from scratch.


Based in the IOS ecosystem, this UI Kit is a mutual effort and has been purposely designed for multi-project use, regardless of subject. And I will be adding more screens and UI elements to the kit in updates.
